# API System

Core backend service providing REST APIs for scheduling, optimization, and data management.

## Endpoints

- `/api/schedules` - Schedule management
- `/api/technicians` - Technician availability
- `/api/resources` - Resource allocation
- `/api/optimize` - ML-based optimization
