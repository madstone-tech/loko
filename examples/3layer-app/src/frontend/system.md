# Frontend System

The Frontend system provides the user interface for the e-commerce platform. It's a React-based single-page application (SPA) that communicates with the Backend API.

## Features

- Product browsing and search
- Shopping cart management
- User authentication
- Order placement and tracking

## Technology

- React 18 with TypeScript
- Redux Toolkit for state management
- React Router for navigation
- Tailwind CSS for styling
